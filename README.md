## Installation

### 1. Create a virtual environment

`python3 -m venv venv`

### 2. Activate the virtual environment

`source venv/bin/activate`

### 3. Install this package in editable mode

`pip install --editable .`

### 4. You're done! Use the package

`hello --help`
